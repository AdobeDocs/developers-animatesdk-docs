## tools.getKeyDown()

#### Availability

Flash MX 2004.

#### Usage

tools.getKeyDown()

#### Parameters

None.

#### Returns

The integer value of the key.

#### Description

Method; returns the most recently pressed key.

#### Example

The following example displays the integer value of the most recently pressed key:
```javascript
var theKey = fl.tools.getKeyDown(); fl.trace(theKey);
```
