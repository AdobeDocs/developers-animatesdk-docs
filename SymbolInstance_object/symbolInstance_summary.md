## symbolInstance summary

> **Inheritance** [Element object](#_bookmark374) \> [Instance object](#_bookmark653) \> SymbolInstance object

#### Availability

> Flash MX 2004.

#### Description

> SymbolInstance is a subclass of the Instance object and represents a symbol in a frame (see [Instance object](#_bookmark653)).
