## library.selectNone()

#### Availability

Flash MX 2004.

#### Usage

library.selectNone()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; deselects all the library items.

#### Example

```
The following examples deselect all the items in the library:
fl.getDocumentDOM().library.selectNone(); fl.getDocumentDOM().library.selectAll(false);

```
