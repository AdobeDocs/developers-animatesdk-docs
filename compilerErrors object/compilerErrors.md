## compilerErrors.clear()

#### Availability

> Flash CS3 Professional.

#### Usage

> compilerErrors.clear()

#### Parameters

> None.

#### Returns

> Nothing.

#### Description

> Method; clears the contents of the Compiler Errors panel.

#### Example

> The following example clears the contents of the Compiler Errors panel:
>
> fl.compilerErrors.clear();
