## TextureAtlasExporter.filePath

#### Availability

Animate 2020.

#### Usage

TextureAtlasExporter.filePath

#### Description

property; It's the location where the texture atlas files needs to be exported.

#### Example

``` javascript
var exporter = new TextureAtlasExporter;
exporter.filePath  = "file:///Downloads/TextureAtlasExporter";
````
