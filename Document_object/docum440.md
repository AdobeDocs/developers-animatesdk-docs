## document.selectNone()

#### Availability

Flash MX 2004.

#### Usage

document.selectNone()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; deselects any selected items.

#### Example

```javascript
The following example deselects any items that are selected:
fl.getDocumentDOM().selectNone();

```
