## document.unlockAllElements()

#### Availability

Flash MX 2004.

#### Usage

document.unlockAllElements()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; unlocks all locked elements on the currently selected frame.

#### Example

The following example unlocks all locked objects on the current frame:

```javascript
fl.getDocumentDOM().unlockAllElements();

```
