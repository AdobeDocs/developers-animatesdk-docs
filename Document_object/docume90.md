## document.group()

#### Availability

Flash MX 2004.

#### Usage

document.group()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; converts the current selection to a group.

#### Example

The following example converts the objects in the current selection to a group:

```javascript
fl.getDocumentDOM().group();

```
