## camera.tintEnabled	

#### Availability

Animate 2019

#### Usage

camera.tintEnabled	

#### Type

Boolean

#### Description

Used to Enable/Disable tint. The value true is if camera is enabled otherwise false.

#### Example

The following example enables the tint:

```javascript
var timeline = an.getDocumentDOM().getTimeline();

timeline.camera.tintEnabled = true;

```
