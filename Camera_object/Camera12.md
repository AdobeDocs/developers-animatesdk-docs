## resetRotation()

#### Availability

Adobe Animate 2019.

#### Usage

camera.resetRotation(frameIndex)

#### Parameters

frameIndex:int

#### Return

Nothing.

#### Description

Property; Reset camera angle to zero.

#### Example

The following example resets camera angle to zero.
```javascript
var timeline = an.getDocumentDOM().getTimeline();
timeline.camera.resetRotation(0);

```
