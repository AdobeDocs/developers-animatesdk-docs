## frame summary

#### Availability

Flash MX 2004.

#### Description

The Frame object represents frames in the layer.

#### Property summary

The following properties can be used with the Frame object:

| **Property**                                   | **Description**                                                                                                                                                                                           |
|------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [frame.actionScript](#_bookmark600)            | A string representing ActionScript code.                                                                                                                                                                  |
| [frame.duration](#_bookmark601)                | Read-only; an integer that represents the number of frames in a frame sequence.                                                                                                                           |
| [frame.elements](#_bookmark603)                | Read-only; an array of Element objects (see [Element object](#_bookmark374)).                                                                                                                             |
| [frame.hasCustomEase](#_bookmark609)           | A Boolean value that specifies whether the frame gets its ease information from the custom ease curve.                                                                                                    |
| [frame.labelType](#_bookmark614)               | A string that specifies the type of Frame name.                                                                                                                                                           |
| [frame.motionTweenOrientToPath](#_bookmark615) | A Boolean value that specifies whether or not the tweened element rotates the element as it moves along a path to maintain its angle with respect to each point on the path.                              |
| [frame.motionTweenRotate](#_bookmark616)       | A string that specifies how the tweened element rotates.                                                                                                                                                  |
| [frame.motionTweenRotateTimes](#_bookmark617)  | An integer that specifies the number of times the tweened element rotates between the starting keyframe and the next keyframe.                                                                            |
| [frame.motionTweenScale](#_bookmark618)        | A Boolean value that specifies whether the tweened element scales to the size of the object in the following keyframe, increasing its size with each frame in the tween (true), or doesn’t scale (false). |
| [frame.motionTweenSnap](#_bookmark619)         | A Boolean value that specifies whether the tweened element automatically snaps to the nearest point on the motion guide layer associated with this frame’s layer (true) or not (false).                   |
| [frame.motionTweenSync](#_bookmark620)         | A Boolean value that if set to true, synchronizes the animation of the tweened object with the main timeline.                                                                                             |
| [frame.name](#_bookmark621)                    | A string that specifies the name of the frame.                                                                                                                                                            |
| [frame.shapeTweenBlend](#_bookmark628)         | A string that specifies how a shape tween is blended between the shape in the keyframe at the start of the tween and the shape in the following keyframe.                                                 |
| [frame.soundEffect](#_bookmark629)             | A string that specifies effects for a sound that is attached directly to a frame ([frame.soundLibraryItem](#_bookmark630)).                                                                               |
| [frame.soundLibraryItem](#_bookmark630)        | A library item (see [SoundItem object](#_bookmark827)) used to create a sound.                                                                                                                            |
| [frame.soundLoop](#_bookmark632)               | An integer value that specifies the number of times a sound that is attached directly to a frame ([frame.soundLibraryItem](#_bookmark630)) plays.                                                         |
| [frame.soundLoopMode](#_bookmark633)           | A string that specifies whether a sound that is attached directly to a frame ([frame.soundLibraryItem](#_bookmark630)) should play a specific number of times or loop indefinitely.                       |
| [frame.soundName](#_bookmark634)               | A string that specifies the name of a sound that is attached directly to a frame ([frame.soundLibraryItem](#_bookmark630)), as stored in the library.                                                     |
| [frame.soundSync](#_bookmark635)               | A string that specifies the sync behavior of a sound that is attached directly to a frame ([frame.soundLibraryItem](#_bookmark630)).                                                                      |
| [frame.startFrame](#_bookmark636)              | Read-only; the index of the first frame in a sequence.                                                                                                                                                    |
| [frame.tweenEasing](#_bookmark638)             | An integer that specifies the amount of easing that should be applied to the tweened object.                                                                                                              |
| [frame.tweenInstanceName](#_bookmark639)       | Assings an instance name to the specified motion object.                                                                                                                                                  |
| [frame.tweenType](#_bookmark640)               | A string that specifies the type of tween.                                                                                                                                                                |
| [frame.useSingleEaseCurve](#_bookmark642)      | A Boolean value that specifies whether a single custom ease curve is used for easing information for all properties.                                                                                      |
