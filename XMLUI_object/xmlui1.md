## xmlui.cancel()

#### Availability

Flash MX 2004.

#### Usage

xmlui.cancel()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; closes the current XMLUI dialog box with a cancel state, which is equivalent to the user clicking the Cancel button.
