Adobe Animate 2020.

#### Usage

layer.setZDepthAtFrame(FrameNumber,ZVal)	

#### Parameters

FrameNumber :frame index,
ZVal:int

#### Returns

nothing	

#### Description

Method;

```javascript
fl.getDocumentDOM().getTimeline().layers[0].setZDepthAtFrame(0,900)
```
