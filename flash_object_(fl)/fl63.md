## fl.saveAll()

#### Availability

Flash MX 2004.

#### Usage

fl.saveAll()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; saves all open documents.
If a file has never been saved, the Save As dialog box displays. If a file has not been modified since the last time it was saved, the file isnâ€™t saved. To allow an unsaved or unmodified file to be saved, use [fl.saveDocumentAs()](#!AdobeDocs/developers-animatesdk-docs/master/flash_object_(fl)/fl65.md).

#### Example

```javascript
The following example saves all open documents:
fl.saveAll();

```
