## fl.clearPublishCache()

#### Availability

Flash CS5.5 Professional.

#### Usage

fl.clearPublishCache()

#### Parameters

None.

#### Returns

Nothing.

#### Description

Method; empties the publish cache.

#### Example

```javascript
The following code empties the publish cache:
fl.clearPublishCache()

```
