## toolObj.showTransformHandles()

#### Availability

Flash MX 2004.

#### Usage

toolObj.showTransformHandles(bShow)

#### Parameters

**bShow** A Boolean value that determines whether to show or hide the free transform handles for the current tool (true shows the handles; false hides them).

#### Returns

Nothing.

#### Description

Method; called in the configureTool() method of an extensible toolâ€™s JavaScript file to indicate that the free transform handles should appear when the tool is active. This method is used only when you create extensible tools.
