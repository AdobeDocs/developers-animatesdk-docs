## document.disableAllFilters()

#### Availability

> Flash 8.

#### Usage

> document.disableAllFilters()

#### Parameters

> None.

#### Returns

> Nothing.

#### Description

> Method; disables all filters on the selected objects.

#### Example

> The following example disables all filters on the selected objects:
>
> fl.getDocumentDOM().disableAllFilters();
